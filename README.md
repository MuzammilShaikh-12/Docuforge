# Environment Setup

## Prerequisites

- Node.js v18+
- npm v9+
- Git
- A Liveblocks/Convex account
- Clerk authentication credentials

## Setup Steps

### 1. Clone & Install

```bash
# Clone repository
git clone https://github.com/muzammilshaikh-12/docuforge.git

# Enter project directory
cd docuforge

# Install dependencies
npm install
```

### 2. Launch Development Server

```bash
npm run dev
```

Your application will be running at `http://localhost:3000` âœ¨
